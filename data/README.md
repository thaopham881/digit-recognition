# Data

This directory is reserved for the MNIST handwritten digit dataset and small sample files used during development and testing.

The MNIST dataset contains grayscale images of handwritten digits from 0 to 9. The program will convert these images into black-and-white images using a selected threshold and represent the black pixels as point sets.

Large dataset files should not be stored directly in this repository. Instructions for downloading or loading the MNIST dataset will be added as the project develops.
